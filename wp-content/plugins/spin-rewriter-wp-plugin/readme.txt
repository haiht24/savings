== Description ==

This is the official Spin Rewriter plugin, developed by the Spin Rewriter in-house team. It uses the Spin Rewriter API to spin your old posts and even create brand new posts.

== Installation ==

Simply extract the ZIP contents into the "wp-content/plugins" folder of your WordPress installation.
